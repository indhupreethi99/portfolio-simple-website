# Personal Portfolio Website

A simple and responsive personal portfolio website built using **HTML** and **basic CSS**. This project showcases my personal profile, skills, and contact information through a clean and minimal design.

## Features

* Responsive homepage layout
* Simple and clean UI
* Contact page for communication
* Built with pure HTML and CSS
* Beginner-friendly project structure

## Pages Included

* **Home Page** – Introduction and portfolio overview
* **Contact Page** – Contact details and communication section

## Technologies Used

* HTML5
* CSS3

## Purpose of the Project

This project was created to practice front-end web development basics and improve my understanding of website structure, styling, and responsive design using HTML and CSS.
